## 제작 의도
- 동영상과 그림이 넘쳐나지만, 어휘 수집에 끌림.
- 독후감이 아니라 저자의 표현을 암기하기 위해 자주 보고 듣는 App을 만들고자 함.
+ "더 나은 사람이 되려면, 좋은 루틴을 가져야 한다."는 말에 감명 받아, 루틴 수행 이력 추가.
+ 계획하고 실행하는 습관을 키우기 위해 "해야 할 일들" 기능 추가.
+ "뇌 용량을 키우는 용도"로 암기를 위한 항목 기능 추가.(메모장,영단어,일단어)
+ 검색 기능 추가.

## App 활용
# 좋은 루틴이 내 삶을 나아지게 할것이라 믿고 싶습니다.  

연필과 노트가 사라지고 있지만, 기억하고 싶은 것들은 늘어나고 있어요.  
육체를 건강하게 하는 만큼, 두뇌도 키우고 싶은 바램이 있습니다.  
일상에서 간직하고 싶은 것들을 오래 기억하려면, 자주 다시 볼 수 있어야 겠어요.  
늘 가져다니는 핸드폰에 사심을 담아 기능들을 구현했습니다.  
기록하고, 자주 보고, 자주 듣고, 기억 창고에 쌓여가길 바라는 마음입니다.  

영상과 사진들로는 길게 생각하고 오래 나눌 수 있는 어휘력을 키우기 어려운것 같아요.
글쓴이가 오래 고민했고 내 마음에 와 닿은 단락들을 적당한 때에 소환하고 싶었습니다.  
실천하는 것은 매번 어렵지만, 그럼에도 다독이고 응원하는 마음을 담았습니다.

- 작성한 자료를 백업과 복원이 쉽도록 했습니다. SQLite DB  
- 흥미로운 내용은 친구들과 공유해서 함께 소통하고자 했습니다.  
- 내 기록이 볼모가 되지 않는 앱을 만들고 싶었습니다.
