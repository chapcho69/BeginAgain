# BeginAgain

제작 의도
- 책에서 발견한 아름다운 구절을 수집하기 위함
  자주 보고 듣고 싶을때 도움 되는 App을 만들고자 함.
+ "좋은 사람이 되려면, 좋은 루틴을 가져야 한다."는 말에 감명 받아, 루틴 수행 이력 추가.
+ 시간 관리 습관을 키우기 위해 "해야 할 일들" 기능 추가.
+ "뇌 용량을 키우는 용도"로 암기를 위한 항목 기능 추가.(메모장,영단어,일단어)
+ 검색 기능 추가.

App 활용
- 

나를 키우는 좋은 루틴이 내 삶을 나아지도록 할것이라 믿습니다.  

필기구와 노트가 사라지고 있지만, 기억하고 싶은 것들은 늘어나고 있어요.  
일상에서 기록하는 것들을 기억하려면, 자주 다시 볼 수 있어야 겠어요.  

영상과 짧은 대화만으로는 길게 대화하기 위한 어휘력을 키우기 어려웠습니다.  
글쓴이가 오래 고민했고 내 마음에 와 닿은 단락들을 기억하고 싶었습니다.  
실천하는 것은 매번 어렵지만, 그럼에도 다독여서 계속하도록 응원하고 싶었습니다.  

- 작성한 자료를 백업과 복원이 쉽도록 했습니다. SQLite DB  
- 흥미로운 내용은 친구들과 공유해서 함께 소통하고자 했습니다.
