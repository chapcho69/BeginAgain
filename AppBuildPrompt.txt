[Begin Again 앱 개발 요청 사항 목록]

1. 코드 안정화 및 빌드
   - SelectionActivity.java 내 경고 및 잠재적 오류(NPE) 수정
   - 프로젝트 전체 클린 빌드 및 정상 실행 확인

2. 독서노트 편집 화면 개선
   - 입력창 크기를 화면의 60%로 확대 및 스크롤 기능 적용
   - 상단 힌트 영역에 바이트 기준 실시간 용량 사용률(% 사용 중) 표시

3. 전역 입력 편의성 개선
   - 모든 입력 화면에서 키보드가 올라올 때 커서 위치가 가려지지 않도록 자동 스크롤 및 adjustResize 적용

4. 스플래시(로딩) 화면 커스텀
   - (추가) manbok.mp4 동영상을 백그라운드 무한 반복 재생으로 적용
   - (삭제) 이후 요청에 따라 동영상 배경 제거 및 리소스 완전 삭제

5. 편집 화면 일괄 UI 개선
   - 독서노트, 메모, 암기장, 비밀노트, 할일들 수정 화면의 본문 입력창 높이를 화면의 50%로 고정
   - 내용이 길어도 저장/삭제 버튼에 즉시 접근 가능하도록 스크롤 구조 최적화

6. 앱 아이콘 제작 및 적용
   - 'Begin Again'의 이니셜 B, A를 형상화한 화이트 벡터 아이콘 제작
   - 배경색을 파스텔 블루(#90CAF9)로 설정하여 적용

7. 습관(루틴) 수행 이력 시각화
   - 상세 화면의 이력 표시를 텍스트 대신 '체크 서클(Check Circle)' 아이콘으로 변경
   - 파란색 아이콘으로 한 줄에 15개씩, 최대 100개까지 표시
   - 전체 이력 보기 팝업에 스크롤 기능 추가

8. 통계 및 성장 대시보드 추가
9. 검색 기능 강화: 최근 검색어 이력(삭제 기능 포함) 제공 및 검색 결과 본문 내 검색어 빨간색 강조 표시
   - GitHub 스타일의 루틴 수행 히트맵(Heatmap) 구현 (최근 1년 기록)
   - 히트맵 진입 시 최신 기록 확인을 위해 자동 우측 스크롤 적용
   - 독서노트 누적 통계(읽은 책 수, 기록한 구절 수) 표시
   - 메모 및 암기장 월별 작성 통계(막대 그래프) 추가

9. 시력 보호를 위한 테마 변경
   - 앱 전체의 흰색 배경을 눈이 편안한 연한 녹색조 화이트(#DEF0DE, 흰색:녹색 4:6 혼합)로 변경

10. 릴리스(Release) 빌드 환경 설정
    - 앱 버전을 4.8 (Rev.8)로 업데이트
    - 별도 키스토어 없이 설치 가능하도록 디버그 서명 키를 이용한 Release 빌드 구성
    - 미사용 리소스 정리 및 빌드 오류 해결

11. 메인 화면 벌집 그리드 확장
    - 육각형 슬롯 수를 40개로 확장하여 화면을 가득 채움
    - 기능 버튼들을 첫 줄은 비우고 2번째 줄부터 순차적으로 초기 배치
    - 드래그 앤 드롭을 통한 위치 이동 기능 유지

12. UI 문구 정제
    - 공유 시 문구를 "shared by..."에서 "Reflection from BeginAgain"으로 변경
    - 대시보드 내부에서 불필요하게 중복되는 "통계" 단어 제거 및 명칭 정제

13. "어휘 Map" 기능 추가 및 고도화
    - 독서노트, 메모, 암기장의 텍스트 데이터를 전수 분석하여 핵심 키워드 추출
    - 빈도수에 따라 단어 크기가 달라지는 워드 클라우드(Word Cloud) 형태의 시각화 지도 구현
    - 백그라운드 처리로 대용량 데이터 분석 시에도 성능 유지 및 메인 그리드 연결
    - 글자가 아닌 기호 및 숫자 필터링 강화
    - 핀치 줌(0.5~4.0배) 및 드래그 이동 기능 추가
    - 전체/최근 1개월 필터 버튼 추가

14. 시스템 UI 간섭 방지 및 레이아웃 최적화
    - 주요 화면에 fitsSystemWindows="true" 적용하여 상태 표시줄 및 내비게이션 바와 겹침 방지
    - 하단 광고(AdView)에 여백(Bottom Margin)을 추가하여 시스템 제스처 영역과의 간섭 최소화
    - 메인 화면 벌집 그리드에 상하좌우 이동(Panning) 기능 추가
